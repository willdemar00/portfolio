📋 SOBRE O PROJETO
───────────────────────────────────────────────────────────────────

Este projeto consiste no desenvolvimento de um portfólio profissional
responsivo utilizando HTML5, CSS3 e JavaScript puro, atendendo todos
os requisitos solicitados pela atividade acadêmica.

───────────────────────────────────────────────────────────────────
✅ REQUISITOS ATENDIDOS
───────────────────────────────────────────────────────────────────

1. ✓ Estrutura Semântica HTML5
   - Cabeçalho com menu de navegação
   - Seção "Sobre Mim" com informações profissionais
   - Seção "Meus Projetos" com grid de projetos
   - Rodapé com informações de copyright

2. ✓ Estilização CSS3
   - Uso de Flexbox para layout do header e cards
   - Uso de CSS Grid para organização de projetos e skills
   - Responsividade completa (desktop, tablet, mobile)
   - Design moderno com variáveis CSS e animações

3. ✓ Interatividade JavaScript
   - Array de objetos contendo projetos
   - Renderização dinâmica dos projetos no HTML
   - Menu mobile responsivo
   - Scroll suave entre seções
   - Animações ao scroll

4. ✓ Versionamento Git
   - Projeto versionado com Git
   - Commits organizados e descritivos
   - Repositório público no GitHub

5. ✓ Publicação GitHub Pages
   - Site publicado e acessível online
   - Configuração correta do GitHub Pages
