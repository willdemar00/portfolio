# Portfólio Profissional

Portfólio profissional desenvolvido como trabalho acadêmico para apresentar habilidades, projetos e informações de contato.

## 🚀 Tecnologias

- **HTML5** — estrutura semântica e acessível
- **CSS3** — design responsivo, variáveis CSS, animações
- **JavaScript** — interatividade, validação de formulário, observadores de interseção

## 📄 Estrutura

```
portfolio/
├── index.html   # Página principal
├── style.css    # Estilos globais e responsividade
├── script.js    # Lógica de interatividade
└── README.md
```

## ✨ Funcionalidades

- Design moderno com tema escuro
- Totalmente responsivo (mobile-first)
- Animações de entrada suaves (Intersection Observer)
- Barras de habilidades animadas
- Formulário de contato com validação client-side
- Navegação com indicador de seção ativa

## 🖥️ Como visualizar

Abra o arquivo `index.html` em qualquer navegador moderno, ou sirva com um servidor estático:

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .
```

Acesse `http://localhost:8080` no navegador.
